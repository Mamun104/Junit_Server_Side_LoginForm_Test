# Junit_Server_Side_LoginForm_Test

# Prerequisites

Install jdk 8 or any LTS version

Configure JAVA_HOME and GRADLE_HOME

Stable internet connection

# How to run this project

Clone the repo

Open cmd in the root folder

Give following commands:

    gradle clean test
